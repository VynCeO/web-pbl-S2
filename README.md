# web-pbl-S2

Website promosi CV menggunakan PHP native.

## Struktur Proyek

- `src/index.php`: Halaman utama CV
- `src/style.css`: Styling untuk website
- `src/script.js`: JavaScript untuk interaktivitas

## Cara Menjalankan

1. Pastikan PHP terinstall.
2. Jalankan server: `cd src && php -S localhost:8000`
3. Buka browser ke `http://localhost:8000`

## Kustomisasi

Edit `index.php` untuk mengubah konten CV sesuai kebutuhan.
